# Stance Ambiguity Pilot

This pilot is the first research-oriented study organized inside the OpenCodebook repository.

The focus is on coding stance, ambiguity, and interpretive uncertainty in GLES open-ended responses. The aim is not to turn the repository into a separate study-specific codebase, but to show how a reusable coding engine can support a concrete empirical project.

At this stage, the pilot includes:

- a study-specific codebook draft
- a reserved data structure for raw and interim GLES files
- notebook placeholders for inspection, sampling, and reliability work
- future workflow stubs in the engine layer
